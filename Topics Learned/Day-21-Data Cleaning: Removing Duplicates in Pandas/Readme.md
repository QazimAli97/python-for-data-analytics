# 🐼 Day 21 – Data Cleaning: Removing Duplicates in Pandas

## 📌 Introduction
Day 21 focuses on an important step in data preprocessing: **removing duplicate records**.

Duplicate data can occur due to system errors, repeated data entry, or data merging issues.

Cleaning duplicate data ensures accurate analysis and reliable insights.

---

## 🎯 Objective
The goal of Day 21 is to:

- Detect duplicate records
- Count duplicate entries
- Remove duplicate rows
- Prepare datasets for analysis

---

## 🧠 Why Removing Duplicates Matters in Data Analytics

In real-world analytics workflows:

- Customer databases may contain repeated records
- Transaction logs may duplicate entries
- Data imports may create duplicate rows

If duplicates are not removed, analysis results can become misleading.

Data cleaning is essential for accurate business insights.

---

## 🔎 Concepts Covered

- `duplicated()` function
- Detecting duplicate rows
- Counting duplicate records
- Removing duplicates using `drop_duplicates()`
- Preparing a clean dataset for analysis

---

## 🧪 Script Summary

The script demonstrates:

- Creating a dataset containing duplicate rows
- Detecting duplicate records
- Counting duplicate entries
- Removing duplicate rows

---

## 💡 Key Learning

Duplicate data can distort analysis results.

Cleaning duplicate records ensures:

- Accurate reporting
- Reliable KPIs
- Correct data aggregation
- Better data quality

Clean data leads to trustworthy insights.
