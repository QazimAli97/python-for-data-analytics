# 🐼 Day 23 – Merging & Joining Data in Pandas

## 📌 Introduction
Day 23 focuses on combining datasets using Pandas.

In real-world analytics, data often comes from multiple tables or sources.  
To perform meaningful analysis, these datasets must be merged.

Pandas provides the `merge()` function which works similarly to SQL joins.

---

## 🎯 Objective
The goal of Day 23 is to:

- Combine datasets using common keys
- Perform INNER JOIN operations
- Perform LEFT JOIN operations
- Understand how relational data works in Pandas

---

## 🧠 Why Merging Data Matters in Data Analytics

In real-world analytics workflows:

- Customer data is stored in one table
- Orders are stored in another table
- Product information is stored separately

To generate insights, analysts must combine these datasets.

Merging datasets is similar to performing SQL JOIN operations.
