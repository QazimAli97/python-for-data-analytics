# 🐼 Day 22 – Data Transformation in Pandas

## 📌 Introduction
Day 22 focuses on transforming data using Pandas.

Data transformation involves modifying, mapping, or restructuring dataset values to prepare them for analysis.

This is a key step in data preprocessing.

---

## 🎯 Objective
The goal of Day 22 is to:

- Create calculated columns
- Transform data using functions
- Map coded values to meaningful labels
- Replace existing values
- Rename dataset columns

---

## 🧠 Why Data Transformation Matters in Data Analytics

In real-world workflows:

- Codes must be converted into readable labels
- Business metrics need to be calculated
- Dataset columns must be standardized
- Values must be cleaned or replaced

Data transformation makes datasets easier to understand and analyze.

---

## 🔎 Concepts Covered

- `apply()` function
- `lambda` functions
- `map()` for value mapping
- `replace()` for modifying values
- `rename()` for renaming columns

---

## 🧪 Script Summary

The script demonstrates:

- Creating a sales dataset
- Calculating revenue using `apply()`
- Mapping region codes to region names
- Replacing product names
- Renaming dataset columns

This simulates a basic data transformation workflow.

---

## 💡 Key Learning

Data transformation prepares raw datasets for meaningful analysis.

Understanding transformation prepares the foundation for:

- Data cleaning workflows
- Feature engineering
- Data standardization
- Business reporting

Well-structured data leads to clearer insights.
