# 🐼 Day 25 – Multi-Level GroupBy & Advanced Aggregation in Pandas

## 📌 Introduction
Day 25 focuses on grouping data by multiple columns using Pandas.

Multi-level grouping allows analysts to analyze datasets across multiple dimensions, such as product and region simultaneously.

This is commonly used in business performance reports.

---

## 🎯 Objective
The goal of Day 25 is to:

- Group data by multiple columns
- Perform advanced aggregations
- Generate multi-dimensional summaries
- Convert grouped data into structured tables

---

## 🧠 Why Multi-Level Grouping Matters in Data Analytics

In real-world analytics workflows:

- Sales are analyzed by product and region
- Customer purchases are segmented by category and location
- Business performance is evaluated across multiple dimensions

Multi-level grouping enables deeper insights into datasets.

---

## 🔎 Concepts Covered

- `groupby()` with multiple columns
- Multi-dimensional aggregation
- Aggregation functions (`sum`, `mean`)
- Using `agg()` for multiple metrics
- Resetting index using `reset_index()`
