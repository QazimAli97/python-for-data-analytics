# 🐼 Day 28 – Correlation Analysis in Pandas

## 📌 Introduction
Day 28 focuses on **correlation analysis** in Pandas.

Correlation helps measure the relationship between numerical variables.  
It shows how one variable changes in relation to another.

---

## 🎯 Objective
The goal of Day 28 is to:

- Understand relationships between variables
- Calculate correlation between columns
- Analyze how features impact key metrics
- Interpret correlation values

---

## 🧠 Why Correlation Matters in Data Analytics

In real-world analytics workflows:

- Analysts check how price impacts sales
- Businesses analyze how discounts affect revenue
- Relationships between variables help in decision-making
- Data-driven insights are generated from correlations

Correlation helps identify patterns in data.

---

## 🔎 Concepts Covered

- `.corr()` function
- Correlation matrix
- Positive vs negative correlation
- Relationship between variables
- Feature impact analysis

## 🧪 Script Summary

The script demonstrates:

- Creating a dataset with numerical variables
- Calculating revenue

- Identifying relationships between variables

## 💡 Key Learning

Correlation helps analysts understand how variables are related.

Understanding relationships prepares the foundation for:

- Data modeling
- Feature selection
- Predictive analytics
- Business decision-making

Insights come from understanding relationships in data.
